📋 Before You Begin

  • ✅ Windows 10 or later (64-bit recommended)
  • ✅ Tally Prime or Tally ERP 9 installed
  • ✅ At least one company loaded in Tally
  • ✅ Active internet connection
  • ✅ Admin rights to install software
  • ✅ Suvidha AI account (sign up free)
1

Download the Connector

Download the Suvidha AI Tally Connector application. This is a lightweight Windows utility (~15 MB) that runs in the background and bridges your Tally data securely.

⬇️ Download Connector (.exe)
💡
64-bit by default. The download is a 64-bit application. If you're using a 32-bit system, contact us at support@complianceai.in for the 32-bit version.
2

Install the Application

  1. Locate the downloaded file — usually in your Downloads folder
  2. Double-click Suvidha AI-Connector-Setup.exe to start installation
  3. If Windows SmartScreen appears, click "More info" → "Run anyway"
  4. Follow the on-screen installer — click "Next" through each step
  5. Wait for installation to complete (usually under 30 seconds)
  6. Click "Finish" — the connector will launch automatically
⚠️
Windows Firewall: If prompted, allow the connector through Windows Firewall. It only communicates between Tally (localhost) and our cloud servers — no external data leaves your machine without sync commands.
3

Configure Tally for Connector Access

Tally needs to be configured to accept connections on its HTTP port. Most recent Tally Prime versions have this enabled by default, but verify with these steps:

  1. Open Tally Prime (or ERP 9)
  2. Press F12 (Configure) or go to F1 → Settings
  3. Navigate to Connectivity settings
  4. Set "Enable ODBC Server" to Yes
  5. Set "Port" to 9000 (default)
  6. Accept and save the settings
Setting Value Notes
Tally HTTP Port 9000 Default port — do not change unless it conflicts
ODBC Server Enabled Required for external connections
Company Must be loaded Open and select the company before syncing
ℹ️
Tally ERP 9 users: Go to Gateway of Tally → F12 → Advanced Configuration and set "Enable Tally.NET Server" to Yes.
4

Launch and Connect

  1. Open the Suvidha AI Connector from your Desktop or Start Menu
  2. Log in with your Suvidha AI credentials (same email/password you use on the website)
  3. The connector will automatically scan for Tally running on localhost:9000
  4. Once detected, you'll see a green "Connected" status with your company name
  5. The connector will now sync data in the background
✅
Success! Once connected, the connector runs silently in the system tray. Return to Suvidha AI website — your Tally data will appear in the integrations panel.
5

Verify the Connection

Once the connector is running, verify the connection from the Suvidha AI website:

  1. Go to Integrations page
  2. In the Tally Integration card, the status should show ● Connected
  3. Your company name will appear next to the status
  4. Ledgers, vouchers, and trial balances will start appearing in your workspace

🔄 Daily Usage

Once setup is complete, your daily workflow is simple:

  1. Open Tally and load the company you want to work with
  2. Launch the Connector (or it auto-starts if set to run on Windows startup)
  3. Use Suvidha AI — data syncs automatically in the background
  4. Ledgers, vouchers, and reports refresh whenever you open a client workspace
💡
Pro tip: Set the connector to run on Windows startup so you never have to manually launch it. Right-click the system tray icon → Settings → "Start on boot".

❓ Frequently Asked Questions

Tally is running but the connector says "Not Connected" ▼
Check the port: Make sure Tally is configured to use port 9000. Go to Tally → F12 → Connectivity and verify the port number. Also ensure that no firewall or antivirus is blocking the connection on localhost:9000.
Can I sync multiple companies? ▼
Yes! The connector automatically detects whichever company is currently loaded in Tally. To sync a different company, simply close the current one and open the desired company in Tally. The connector will detect the change and sync accordingly.
Is my data secure? What data leaves my computer? ▼
The connector communicates locally with Tally (localhost only) and sends encrypted data over HTTPS to Suvidha AI servers. Only the specific data required for compliance (ledgers, vouchers, trial balance) is synced. No passwords or sensitive credentials leave your machine. All data is encrypted in transit and at rest.
Does the connector work with Tally running on a different PC in the network? ▼
By default, the connector connects to Tally on the same machine (localhost:9000). If Tally is running on a different machine in your LAN, you can change the host in the connector settings to the IP address of that machine (e.g., 192.168.1.100:9000).
I'm using a 32-bit system. Can I still use the connector? ▼
The default download is a 64-bit application. For 32-bit systems, contact us at support@complianceai.in and we'll send you the 32-bit version download link.
How do I uninstall the connector? ▼
Go to Windows Settings → Apps → Apps & features, search for "Suvidha AI Connector", and click Uninstall. The connector will be completely removed without leaving any traces.
