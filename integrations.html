Tally Prime / ERP 9 Integration

Sync ledgers, vouchers, and trial balances. Choose your preferred connection method.

Not Connected

Keep using this website. Install a small utility to bridge the connection to Tally on your machine.

RECOMMENDED

2 Desktop App

Download the full ComplianceAI desktop app. No setup required. Sync works when Tally is open.

✅ Zero Config · ✅ Enhanced Security

Ensure Tally is running on port 9000

1 Using the Connector

  1. Open Tally Prime or ERP 9 and ensure your company is loaded.
  2. Run the ComplianceAI Connector (.exe).
  3. Paste the Connection Token when asked.
  4. The Connector will detect Tally and start syncing automatically.

2 Using Desktop App (Better)

  1. Download and install the full Desktop App.
  2. Log in with your email and password.
  3. Keep Tally open in the background.
  4. Click "Sync Tally" in the app dashboard. No token needed!
