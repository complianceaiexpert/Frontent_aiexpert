Step-by-step Setup
1

Download the Suvidha AI Connector

Download and install the small connector utility on the same computer where Tally is installed. This bridges the connection between Tally and Suvidha AI.

2

Open Tally and load your company

Launch Tally Prime or Tally ERP 9 on your computer. Make sure the company you want to sync is loaded and active in Tally.

3

Run the Connector

Open the Suvidha AI Connector app you installed in Step 1. It will automatically detect Tally running on localhost:9000. Keep it running in the background.

4

Test the connection

Click the "Test Connection" button above. If everything is set up correctly, the status will turn green and show your company name.

5

You're all set!

Go to the Data Entry service and start working. Your ledgers, vouchers, and trial balances will sync automatically whenever Tally and the Connector are running.

Frequently Asked Questions

Why am I seeing "Bridge not running" error?

This means the Connector app is not running or Tally is not open. Check these:
  • Make sure Tally is open with a company loaded
  • Make sure the Suvidha AI Connector is running (check your system tray)
  • Tally must be accessible on localhost:9000 — this is the default port
  • If you changed Tally's port, update it in the Connector settings

What if Tally says "port already in use"?

Another application is using port 9000. To fix this:
  • Close any other app using port 9000 (like another Tally instance)
  • Or change Tally's server port: Go to Tally → F12 → Advanced Configuration → Set Client/Server port
  • Then update the same port in the Connector settings

Why is my company name not showing after connecting?

The connection is working but Tally might not have a company loaded:
  • Open Tally and go to Gateway of Tally
  • Select and open the company you want to sync
  • Then click "Test Connection" again

Why won't the Connector app start after installing?

Try these steps:
  • Right-click the Connector and select "Run as Administrator"
  • Check if Windows Defender or antivirus is blocking it — add an exception
  • Re-download and re-install the Connector
  • Make sure you're on Windows 10 or later

Why is my data not syncing even though I'm connected?

If the connection shows green but data isn't appearing:
  • Make sure the correct company is loaded in Tally
  • Try refreshing the page and navigating to Data Entry again
  • Check that Tally's ODBC/XML Server is enabled (Tally → F12 → Advanced Config → Enable ODBC Server = Yes)
  • Restart both Tally and the Connector

Can I use Suvidha AI on Mac or Linux?

Tally itself only runs on Windows. However, you can access Suvidha AI from any browser (Mac, Linux, iPad). The Connector and Tally just need to be running on any Windows machine on your network — it doesn't have to be the same computer you're browsing from.
